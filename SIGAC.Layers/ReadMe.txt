ReadMe SIGAC.Layers:

Capas: 

-Data: 
Aqui solo se encuentran clases referentes al proceso de extraer informacion de las diferentes fuentes usadas en la aplicacion.

-Application: 
Aqui solo se encontraran clases referentes a cualquier proceso interno que necesite realizar la aplicacion para funcionar.

-Bussiness: 
Aqui solo se encontraran clases referentes a cada uno de los models y viewmodels que necesitemos dentro de la aplicacion.



Capa de Datos: 

-FtpContext: 

Aqui estan los metodos, procedimientos, funciones y propiedades referentes al envio de archivos a traves del protocolo ftp.

-DbContext: 

Aqui estan los metodos, procedimientos, funciones y propiedades referentes a la manipulacion de datos a traves de los proveedores de base de datos usados en el sistema
		-Oracle DB
